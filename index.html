I then ran some tweaks though the powershell utility.

If you need to search Microsoft Windows click start, or press ⊞+S or press the ⊞ key, then begin typing.

Backups

Back up and Restore

  • Back up and Restore (doesn't require Microsoft account) can be found by:
    • Pressing ⊞+S
    • Search for Control Panel
    • Control Panel -> System and Security -> Back up and Restore (Windows 7).
  • I have created a System Image backup of Windows on 27/02/2025 it can be found at:
    • Secondary Disk D:\WindowsImageBackup\
  • You can return to this System Image backup at any time using 'Back up and Restore (Windows 7)' if anything in windows breaks badly or you want a "fresh install". Just make sure to backup all your files on a external drive first.
  • I have also setup Windows Backup this will backup all your files and the system state every 2 weeks on Sunday at 7:00pm

Windows Back Up - Shouldn't need use but might be useful to have a backup on OneDrive just in case your laptop is stolen

  • Windows Back Up (requires Microsoft account - I would recommend against using your university email as you may be locked out of this when you graduate) can be found by:
    • Pressing ⊞+S
    • Search for Windows Back Up
  • Or open Backup and Restore in the system Settings.

winget

I've installed a few things using the winget command in the Terminal.
winget is very useful for installing a wide variety programs and applications (packages).

Using 'UniGetUi' to interface with winget

I have installed the program 'UniGetUI' to enable the easy installation and updating of winget packages.

  • 'UniGetUi' can be found in the 'All' list in the 'Start Menu' or by searching the 'Start Menu'.

To install a winget package.

  1. Navigate to the 'Discover Packages' tab on the left.
  2. You can then search and install new winget packages.
  3. It will ask you for your password or fingerprint for each package install.

To update any packages installed through winget.

  1. Navigate to the 'Software Updates' tab on the left.
  2. Then click update selected packages.
  3. It will ask you for your password or fingerprint for each package update.

Using winget in the terminal

  1. Press ⊞+X or right click the start menu
  2. Click Terminal (admin)
  3. Enter your password or use your fingerprint
  4. Type winget install (package) replace (package) with any application or program you want to install

Example

Install neofetch (A package to show 'About This PC' in the terminal)

winget install neofetch

Issues with Microsoft Store

Due to removing some bloatware installation some software using the Microsoft Store fails with Code: 0x80040154

  • If this happens try to check the internet for an alternative way to install the package.
  • Else use winget, either via UniGetUI or in the Terminal (admin).

Example

Installing Teams from the Microsoft Store using winget in the Terminal.

winget install teams
